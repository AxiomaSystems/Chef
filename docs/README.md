# Preppie Docs

This folder contains the current product, architecture, and engineering documentation for Preppie.

The definitive product and business source is `business_plan.pdf`. Markdown docs should summarize or operationalize that plan, not contradict it.

## Start Here

- `business_plan.pdf` - definitive product, business, pricing, GTM, roadmap, risk, and financial plan.
- `business.md` - Markdown business summary derived from the plan.
- `goals.md` - product goals and implementation priorities.
- `FEATURES.md` - customer-facing feature inventory.
- `architecture.md` - implemented system architecture and current boundaries.
- `models.md` - conceptual model map for shared/API contracts.
- `decisions.md` - key engineering and product decisions.
- `branching.md` - current branch and merge policy.
- `future-preppie-brand-transition.md` - naming guidance for internal Chef references that have not been renamed yet.
