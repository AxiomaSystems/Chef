Visual quick-start guide

How to Use Butter Me

Start on Home, fill your inventory by voice, choose or create recipes, build a cart for what you lack, then cook hands-free when everything is ready.

Start Here: Home Inventory Recipes Cart Shopping Cook hands-free
1 Home

Start at the Home Page

Use the Home screen as the launch pad. The profile icon is also visible here.

Start Here
Search or tap a card
2 Voice Add

Add Things to Inventory

Open Your Kitchen, tap Voice Add, record items, review the transcript, then save.

Record Save
3 Recipes

Select or Create a Recipe

Browse Public, Mine, or Saved. Use the plus menu for URL import or a new recipe.

New recipe
Add recipe from URL Create new recipe
4 Check

Check Inventory and Select

Open a recipe, check missing ingredients, then add one or multiple recipes to the cart.

Check against inventory Generate Cart
5 Shopping

Buy What You Lack

Create a shopping list from the cart, checkout, receive items, then come back to cooking.

Create shopping list Checkout
6 Cook

Cook Hands-Free

When ingredients are ready, start hands-free mode. Say "Chef" when you need help.

Quick Tips

1 Inventory first makes missing-ingredient shopping more accurate.
2 Use the orange circled controls as the next action to tap.
