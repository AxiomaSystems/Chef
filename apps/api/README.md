# API

This app is the NestJS backend for Cart Generator.

## Current State

- starter NestJS application
- default root endpoint
- starter unit and e2e tests

## Commands

```bash
pnpm start
pnpm start:dev
pnpm start:prod
pnpm test -- --runInBand
pnpm test:e2e -- --runInBand
pnpm build
```
